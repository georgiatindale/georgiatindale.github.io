# Georgia Tindale - Portfolio Website

Live site: https://georgiatindale.com

## Local Development

To edit and preview this site locally with live reload:

### Using VS Code Live Server Extension (Recommended)

1. Install the Live Server extension:
   - Open Extensions view (Ctrl+Shift+X)
   - Search for "Live Server" by Ritwick Dey
   - Click Install

2. Open the project folder in VS Code

3. Right-click on `index.html` and select "Open with Live Server"
   - The site will open in your browser
   - Changes to HTML, CSS, and JS files will automatically refresh the browser
